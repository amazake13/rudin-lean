Let \(X\) be a topological vector space over \(\mathbb {K}\). For each \(a \in X\) the translation \(T_a\colon x \mapsto x + a\) is a homeomorphism of \(X\) onto itself, and for each nonzero scalar \(c\) the map \(M_c\colon x \mapsto cx\) is a homeomorphism as well.

For a set \(s \subseteq E\) in a real topological vector space, the Minkowski functional of \(s\) is \(\mu _s(x) = \inf \{ t {\gt} 0 : x \in t s \} \).
